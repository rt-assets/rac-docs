## About

RAC (Runtime Advanced Cables) is a Verlet-based cable and rope toolkit for Unreal Engine. Physics simulation, surface draping, player interaction, tethering, riding, swinging, climbing, and towing - all from Blueprint.

- **Publisher:** [RT Assets](https://www.fab.com/sellers/RT%20Assets)
- **Engine Support:** UE 5.4 - 5.7
- **Contact:** rtassets@proton.me
